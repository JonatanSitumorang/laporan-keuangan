Q: Aplikasi ini gratis?
A: 100% GRATIS! Tidak ada biaya, iklan, atau subscription.
   Anda cuma perlu web browser yang paling basic.

Q: Perlu install?
A: TIDAK! Buka aja di browser. Langsung jalan.
   Tidak perlu download, install, setup apapun.

Q: Aman tidak? Privacy gimana?
A: 100% AMAN! Data disimpan di komputer Anda sendiri.
   Tidak ada cloud, server, atau upload ke internet.
   Hanya Anda yang tahu pengeluaran Anda.

Q: Bisa buat backup?
A: BISA! Lihat PANDUAN_CEPAT.txt section "Export Data".
   Copy data via browser console, save ke file.

Q: Kalau browser ditutup, data hilang?
A: TIDAK! Data tersimpan permanent di Local Storage.
   Buka browser lagi, data masih ada.

Q: Bisa edit pengeluaran yang sudah input?
A: Untuk sekarang: hapus & input ulang.
   Future feature: edit option akan ditambah.

Q: Kalau ada bug/error?
A: Buka F12 (DevTools), lihat Console untuk error message.
   Lihat PANDUAN_CEPAT.txt section "Troubleshooting".

Q: Bisa share data dengan keluarga?
A: Data tersimpan di browser saja, tidak bisa direct share.
   Workaround: Export data, share file ke device lain.

Q: Support Android/iPhone?
A: BISA! Website responsive, bisa buka di smartphone.
   Data per device terpisah (tidak sync cross-device).

Q: Gimana kalo cache browser dihapus?
A: Data akan hilang jika clear cache/storage.
   Tips: Backup regular sesuai PANDUAN_CEPAT.txt.
