💡 TIPS EMAS PENGGUNAAN
═══════════════════════════════════════════════════════════════════════════

TIP #1: Input KONSISTEN
Input setiap hari lebih baik dari 30 input di akhir bulan
→ Lebih mudah ingat detail pengeluaran
→ Data lebih akurat
→ Lebih mudah tracking

TIP #2: Deskripsi JELAS
Bukan: "Makan", "Belanja", "Kendaraan"
Lebih baik: "Sarapan nasi kucing", "Belanja di Indomaret", "Bensin motor"
→ Lebih mudah recall
→ Better insights

TIP #3: Kategori KONSISTEN
Pilih yang paling sesuai, jangan tebakan
→ Data lebih reliable
→ Grafik lebih akurat

TIP #4: Review MINGGUAN
Setiap minggu ambil 5 menit:
- Review pengeluaran minggu lalu
- Lihat kategori mana paling boros
- Plan pengeluaran minggu depan
→ Lebih aware dan mindful

TIP #5: Backup REGULAR
Setiap akhir bulan backup data
→ Data terproteksi
→ Bisa restore jika browser error

═══════════════════════════════════════════════════════════════════════════
🔧 TROUBLESHOOTING CEPAT
═══════════════════════════════════════════════════════════════════════════

MASALAH: Website tidak bisa dibuka
SOLUSI:
1. Pastikan file index.html ada di folder ini
2. Coba buka dengan browser lain (Chrome, Firefox, Edge)
3. Restart komputer
4. Buka DevTools (F12) → Console, lihat error message

MASALAH: Data tidak tersimpan
SOLUSI:
1. Periksa Local Storage: F12 → Application → Local Storage
2. Pastikan browser tidak dalam private/incognito mode
3. Clear cache & coba lagi
4. Gunakan browser yang berbeda

MASALAH: Grafik tidak muncul
SOLUSI:
1. Pastikan sudah ada data input
2. Refresh halaman (F5 atau Ctrl+R)
3. Buka DevTools (F12) → Console, cek error
4. Coba di browser lain

MASALAH: Tombol tidak berfungsi
SOLUSI:
1. Refresh halaman
2. Clear cache browser
3. Buka DevTools (F12) → Console
4. Coba di browser lain

Untuk Q&A lebih lengkap → baca PANDUAN_CEPAT.txt
